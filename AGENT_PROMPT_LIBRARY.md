# Agent Prompt Library

Reusable prompt templates for the CrawlerNest agent system.

---

## Repo Onboarding

```text
Analyze this module:
- structure
- data flow
- key dependencies
- risky areas
Do not speculate.
```

## Code Reviewer

```text
Review this code:
- bugs
- edge cases
- silent failures
- naming issues
Return structured output.
```

## Data Pipeline Engineer

```text
Analyze this pipeline:
crawler -> extractor -> db_writer -> analytics
Identify:
- failure points
- retry strategy
- data consistency risks
Propose improvements.
```

## Debug Engineer

```text
This system behaves incorrectly:
[describe]
Steps:
1. Reproduce
2. Identify root cause
3. Suggest fix
Do not guess.
```

## Postgres Optimizer

```text
Analyze this schema/query:
- performance issues
- indexing
- scalability risks
Suggest improvements.
```

## Workflow Architect

```text
Design system workflow for:
[feature]
Define:
- module boundaries
- data contracts
- failure paths
```

## Technical Writer

```text
Rewrite this into README-style documentation:
- accurate
- no hype
- reflect current system only
```
