# Antigravity Integration Skeleton

This directory stores repo-tracked skill templates for Gemini Antigravity-style setups.

Target runtime path example:

```text
~/.gemini/antigravity/skills/
```

Recommended workflow:

1. edit and version the skill here
2. copy or sync it into the real Antigravity skills directory on your machine
3. keep wording aligned with the canonical `agents/` prompt
